    VBA: Check if String exists within collection.

    Hello.

    I have two collections of Strings. I need to see if a certain String exists in these collections. This is what I have, but it doesn't work and it's also inefficient in terms of speed. Is there a better approach? Thank you for your help.

    -Sir Phoenix

    Code:
    Public Function DoesItemExist(mySet As Collection, myCheck As String) As Boolean
    On Error Resume Next
    
    Dim myNum As Integer
        
    DoesItemExist = False
    For myNum = 0 To mySet.Count - 1
        If myCheck = mySet(myNum) Then
            DoesItemExist = True
        End If
    Next
    On Error GoTo 0
End Function

    Code:
    Public Function DoesItemExist(mySet As Collection, myCheck As String) As Boolean
Dim myNum As Integer
    DoesItemExist = False
    For myNum = 1 To mySet.Count
        If myCheck = mySet(myNum) Then
            DoesItemExist = True
            Exit Function
        End If
    Next
End Function
    Or

    Code:
    Public Function DoesItemExist(mySet As Collection, myCheck As String) As Boolean
    DoesItemExist = False
    For Each elm In mySet
        If myCheck = elm Then
            DoesItemExist = True
            Exit Function
        End If
    Next
End Function

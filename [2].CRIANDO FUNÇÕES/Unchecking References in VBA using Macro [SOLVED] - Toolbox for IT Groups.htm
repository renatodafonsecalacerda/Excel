    Unchecking References in VBA using Macro
    pyniu asked Apr 12, 2012 | Replies (1)
    Hi
    I made some woorkbook with vba macros which use references:
    - Visual Basic for Applications
    - Microsoft Excel 14.0 Object Library
    - OLE Automation
    - Microsoft Office 14.0 Object Library

    But after installing MS silverlight on OS, reference is automatically adding to list and is active then. This completely break down some objects and macros are useless.


    Is there any way to strictly define which references needs to be used?
    Or how to unselect 'MS silverlight' reference from list using macro?
    (i'm beginner:)

    In order to remove a specified reference from your project, you can use
    the following procedure:
    '--
    Sub RemoveSpecifiedReference()
    ' This example removes 'PowerPoint' reference
    ' from ThisWorkbook.VBProject .
    ' Instead of 'ThisWorkbook' you can work with ActiveWorkbook.
    '--
    Dim ref As Variant
    Dim i As Integer
    '--
    With ThisWorkbook.VBProject
    For i = .References.Count To 1 Step -1
    Set ref = .References(i)
    If Not ref.BuiltIn Then
    'MsgBox ref.Name
    ' Customize Reference name

    If ref.Name = "PowerPoint" Then
    MsgBox ref.Name
    ' uncomment the following code line
    '.References.Remove ref
    End If
    End If
    Next i
    End With
    End Sub
    '--
    Because you don't know the name of reference to be deleted, here is a
    simple procedure thet list all references from your vb project in
    worksheet named 'References'.
    (add worksheet 'References' in your workbook and run the following
    procedure:
    '--
    Sub ListReferencePaths()
    ' Purpose: This SAMPLE routine determines full path and Globally Unique
    Identifier (GUID)
    'to each referenced library.
    '--
    On Error Resume Next
    Dim i As Long
    With ThisWorkbook.Worksheets("References")
    .Cells.Clear
    .Range("A1") = "Reference name"
    .Range("B1") = "Full path to reference"
    .Range("C1") = "Reference GUID"
    End With
    For i = 1 To ThisWorkbook.VBProject.References.Count
    With ThisWorkbook.VBProject.References(i)
    ThisWorkbook.Sheets("References").Range( "A65536").End(xlUp).Offset(1, 0)
    = .Name
    ThisWorkbook.Sheets("References").Range( "A65536").End(xlUp).Offset(0, 1)
    = .FullPath
    ThisWorkbook.Sheets("References").Range( "A65536").End(xlUp).Offset(0, 2)
    = .GUID
    End With
    Next i
    On Error GoTo 0
    End Sub
    '--
    If you want ro remove broken references from your project, here is a
    procedure that will do it:
    '--
    Sub RemoveBrokenReferences()
    ' Purpose: This example ilustrates how to remove all broken references
    from the VBA project
    '--
    Dim ref As Variant
    Dim i As Integer
    '--
    With ThisWorkbook.VBProject
    For i = .References.Count To 1 Step -1
    Set ref = .References(i)
    If Not ref.BuiltIn Then
    If ref.IsBroken Then
    .References.Remove ref
    End If
    End If
    Next i
    End With
    End Sub
    '--

    Best Regards
    Andrija Vrcan
